## PATZ란?

**PATZ**는 네일아트 예약 경험을 혁신하는 스타트업입니다.

고객은 샵의 아트를 한눈에 보고, 원하는 디자인을 골라 몇 번의 터치만으로 예약을 완료할 수 있습니다. 샵 오너는 별도 앱 설치 없이 웹 대시보드에서 아트 관리, 예약 확인, 고객 소통까지 한 곳에서 처리합니다.

### 핵심 가치

| 고객                  | 샵 오너                   |
| --------------------- | ------------------------- |
| 이달의 아트 피드 탐색 | 아트 카탈로그 손쉽게 관리 |
| 3단계 간편 예약       | 예약 요청 실시간 확인     |
| 카카오 채널 연동 소통 | 노쇼 방지 선입금 시스템   |

### 비즈니스 모델

네일샵에 예약 SaaS를 제공하고, 예약 건당 수수료 또는 월 구독 모델로 수익화합니다. 1인 네일샵부터 다인샵까지, 예약 관리에 소요되는 시간과 노쇼 비용을 줄여줍니다.

### 제품 구성

- **patz-web** (이 저장소) — 샵 오너 대시보드 + 고객 예약 웹앱
- **PATZ 모바일 앱** (예정) — 고객용 네이티브 앱, 동일 Supabase DB 공유

- **Tailwind CSS**
- **Supabase** (Postgres + Auth + Storage)
